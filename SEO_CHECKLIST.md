## ⚠️ Проблемы и решения

### Проблема: Страницы не индексируются
**Решение:**
1. Проверить robots.txt (нет ли запрета)
2. Проверить meta robots (должен быть index, follow)
3. Отправить URL через "Переобход страниц" в Яндекс
4. Запросить индексирование в Google Search Console

### Проблема: Низкие позиции
**Решение:**
1. Увеличить количество контента на страницах
2. Добавить больше ключевых слов в заголовки
3. Получить внешние ссылки с авторитетных сайтов
4. Зарегистрироваться в каталогах (2GIS, Zoon, и т.д.)

### Проблема: Сайт медленно грузится
**Решение:**
1. Проверить в PageSpeed Insights
2. Оптимизировать изображения (WebP)
3. Включить кэширование
4. Использовать CDN для статики
